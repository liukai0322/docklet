** System **
* vim emcas nano
* tmux w3m curl
* zsh
* unrar,p7zip-full,p7zip-rar
* openssh-server
* openbox
* vpnc ( web )
* fonts-droid, fonts-wqy-zenhei, fonts-wqy-microhei, fonts-arphic-gbsn00lp, fonts-arphic-gkai00mp, fonts-arphic-ukai, fonts-arphic-uming latex-cjk-all

* scala

* clisp, guile
* golang
